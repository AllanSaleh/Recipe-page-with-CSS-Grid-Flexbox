# Recipe Grid Flexbox

A simple, responsive recipe page built with HTML and CSS using CSS Grid and Flexbox. The page displays a collection of recipe cards, a sticky header, and an about section, all styled for both desktop and mobile devices.

## Features
- Responsive layout using CSS Grid and Flexbox
- Sticky header and sidebar
- Hamburger menu for mobile navigation
- Modern, clean design
- Easy to customize recipe cards

## Getting Started

1. **Clone or Download the Repository**
	- Clone this repo or download the ZIP and extract it.

2. **Open the Project**
	- Open the folder in your code editor (e.g., VS Code).

3. **View in Browser**
	- Open `index.html` directly in your web browser.
	- No build steps or dependencies are required.

## Project Structure

```
├── index.html        # Main HTML file
├── style.css         # CSS styles
├── assets/           # Images (logo, profile, etc.)
└── README.md         # Project documentation
```

## Customization
- To add or edit recipes, modify the recipe card sections in `index.html`.
- To change images, update the files in the `assets/` folder or use new image URLs.
